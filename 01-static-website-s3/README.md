# 🌐 Static Website Hosting on Amazon S3

## 📌 Project Overview

This project demonstrates the deployment of a static website using Amazon S3 Static Website Hosting.

The project is part of my Cloud Engineering Internship journey and provides hands-on experience with cloud storage, static website deployment, and AWS configuration.

## 🎯 Objectives

- Create a static website.
- Upload website files to Amazon S3.
- Configure static website hosting.
- Deploy and access the website through AWS.
- Practice Git and GitHub version control.
